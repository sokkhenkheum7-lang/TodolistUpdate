# Dashboard Guide – How to Use Each Section

This guide explains every section of the dashboard and how to use it effectively.

## 1. Logo 

The Logo brand section represents your application, workspace, or identity of website.

- Purpose
- Displays the application logo.
- Shows workspace information.
- Provides quick access to the main dashboard.

# How to Use

- Click the logo to return to the Dashboard Home.
- View your current workspace name and details.
- Use it as a navigation anchor when moving between pages.

## 3. New Task Button

The + New Task button allows users to create new tasks quickly.
+ Purpose
- Add new work items.
- Create goals and action plans.
- Increase productivity by organizing tasks.

## How to Use

- Click + New Task.
- Enter task information:
- Title
- Description
- Category
- Priority
- Due Date
- Save the task.
- The task will automatically appear in the Workspace Board.
- Best Practice
- Always assign:
- A category
- A priority level
- A due date
- This helps improve organization and reporting accuracy.

## 4. Hero Section

The Hero Section provides a quick overview of your workspace status.
+ Purpose

- Available Actions

## Browse Task Matrix

- Open the complete task overview.

## Quick Add Goal

- Create goals instantly.

## How to Use

- Read the workspace summary.

## 5. Metrics Cards

- Metrics Cards provide an instant snapshot of workspace performance.
- Available Metrics
- Total Workspace Volume
- Shows the total number of tasks.
- Goals Completed
- Displays completed tasks and goals.
- Pending Workload
- Shows tasks that still require attention.
- Critical Focus
+ Displays urgent or high-priority items.
- Color Indicators
- Color	Meaning
- Blue	Normal
- Yellow	Warning
- Red	Critical
- green color Active

## How to Use

- Review metrics daily.
- Identify bottlenecks quickly.
- Prioritize critical items first.

## 6. Productivity Overview

The Productivity Overview visualizes task distribution across workflow stages.
- Workflow Stages
+ Todo
- Tasks not yet started.
+ Active
- Tasks currently being worked on.
+ Review
+ Tasks awaiting approval or verification.
+ Done
- Completed tasks.
- Features
- Task counts by status.
- Percentage distribution.
- Live updates.
- Progress visualization.

## How to Use

- Monitor workload balance.
- Identify overloaded stages.
- Track productivity trends.
- Ensure tasks continue progressing toward completion.

## Benefits

- Improved visibility.
- Better planning.
- Faster issue detection.
